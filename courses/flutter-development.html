Flutter Development

Flutter is an open-source UI software development kit by Google. It is used to develop cross platform apps from a single codebase for any web browser, Android, iOS, Linux, macOS & Windows. It is known for fast performance, expressive & flexible UI, and hot reload capabilities. It is a popular choice for developing mobile apps, but can also be used to build web apps, desktop & embedded apps.

At Hyreblock, we offer a comprehensive Flutter package for learners that comprises training, projects, help on creating a portfolio & mentorship. Our team with trainers having relevant experience will work with you on the strategic goals of your career during the engagement period.

The course is designed by industry experts who have relevant experience in solving complex business problems using Flutter. Also, there are dedicated sessions to address Q&A or feedback from learners.

  • The course can be conducted on weekends or weekdays (flexible as per your schedule).
  • At the end of this course, all learners will be aware of key Flutter concepts & capabilities.

Course Schedule

To learn from the basics to comprehensive Flutter expertise in 1 month. The course is structured to have both classroom learning and hands-on projects in a cohort-based learning environment.

  • Basics of Flutter and Dart

    • Introduction to Flutter, Flutter and Dart Relationship, Flutter's Multi-Platform Capability, Setting Up Flutter Development Environment, Exploring Different Platform Setups, Introduction to Dart Programming, Material Design and Cupertino Design, Running and Editing Your First Flutter App, Working with Emulators and Simulators

  • Layouts in Flutter

    • Introduction to Widgets in Flutter, Layout Widgets in Detail, Commonly Used Widgets, Applying Decorations and Styling Widgets

  • Prerequisites before Developing Apps

    • Understanding pubspec.yaml, Importing Features from Packages, Basic State Management Concepts, Basic State Management Concepts, main() vs runApp(): Differences and Usage, Positional and Named Arguments in Dart, Alignment, Padding, and Margin Concepts, Cross and Main Axis Alignment Explained

  • Roll A Dice App

    • Generating Random Numbers and Using math Package, Working with Asset Images, Building the Dice Rolling App, Adding Animations

  • To-Do List App

    • Introduction to Data Persistence, Using Hive and SQLite for Data Storage, Creating a TabView for Different Lists, Implementing ListView and Scrolling, Managing Data with JSON and Maps, Adding Checkbox Functionality, Global State Management

  • Marvel Hero App

    • API Integration Basics, Understanding JSON Parsing, Creating Models for Parsed Data, Using HTTP Package or Dio for API Calls, Building the Marvel Hero App

  • Social Media App Using Firebase

    • User Authentication with Firebase, Firestore Database Integration, Implementing Camera Functionality, Uploading Images to Firebase Storage

  • Software Development Skills

    • Version Control with Git, Code Documentation, Design Patterns
